Инструкция

Пошаговая инструкция о том, как принять
учавствие в турнире

1. Перейти в телеграм чат по ссылкеВот тут.

Чат должен называться: "Petya_loh228".

2. Написать: "Хочу учавствовать".
Админ проверит вас и примет решение по принятию.

3. Сказать вид принятия учавствия: "Платный" или через отбор.
Платный вид стоит 70руб, после его приобретения вы без отборов попадаете на турнир и соревнуетесь за 450руб.

4. Указать свой Steam ID
Админ сохранит ваш ID и сразу запишет в список участников.
